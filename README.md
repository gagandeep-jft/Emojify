# Text to Emoji

A simple site for converting text to emojified text

![emojify demo](emojify_demo.gif)

### Supported Translations

```javascript
{
  smile: "😄",
  wink: "😉",
  khushi: "😊",
  pyaar: "🥰",
  dukh: "🥲",
  dard: "🥲",
  peeda: "🥲",
  money: "🤑",
  hugs: "🤗",
  shuu: "🤐",
  haha: "😂",
  ded: "💀",
  dead: "💀",
}
```

### Note

To get the most out of it, zoom-in and enable dark-mode via browser extension
